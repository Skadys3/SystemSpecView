# 🖥️ System Specifications Viewer

### Современное приложение для мониторинга характеристик компьютера в реальном времени

Мониторинг процессора, памяти, дисков, сети, видеокарты, материнской платы и процессов через красивый интерфейс на **CustomTkinter**.

---

# ✨ Возможности

* 📊 Современный Dashboard
* ⚡ Мониторинг CPU в реальном времени
* 🧠 Использование оперативной памяти
* 💾 Контроль дисков
* 🌐 Мониторинг сети
* 🎮 Информация о видеокарте
* 🖥 Сведения о материнской плате
* 🔋 Информация о батарее
* 📋 Встроенный диспетчер задач
* 🌙 Светлая и тёмная темы
* 📈 История нагрузки (Sparkline)
* 📄 Экспорт отчёта
* ⚙️ Автоматическое обновление данных

# 🚀 Быстрый старт

```bash
git clone https://github.com/USERNAME/system-specifications-viewer.git

cd system-specifications-viewer

pip install -r requirements.txt

python main.py
```
